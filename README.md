# EcoAI Public Review

This repository contains the public review versions of the **EcoAI** website concept.

## Main review versions
- `v5/` — clean and elegant direction
- `v6/` — branding exploration with alternate logo treatment

## Archive
Earlier concept explorations are stored in:
- `archive/`

## Shared assets
- `assets/` — shared logo and image assets used across versions

## Recommended review flow
If the team wants to compare the latest directions, start with:
1. `v5/`
2. `v6/`

## How to preview locally
Open directly in a browser:
- `v5/index.html`
- `v6/index.html`

Or run:
- `v5/preview.bat`
- `v6/preview.bat`

## Folder structure
- `v5/` → main refined review version
- `v6/` → alternate branding direction
- `archive/` → previous exploration versions
- `assets/` → shared assets

## Notes
- The current review is intentionally lightweight and easy to modify.
- Dropdown interactions have been added to support future page-level navigation.
- Archive versions are retained for design comparison and iteration history.
